Projects

My most notable project was the one that I created during my computer science A-level, "Moview"

Moview was created in python and was created with the intention of getting movie reviews through the use of tweets

These tweets would go through sentiment analysis to convert those tweets into positive or negative. At which point the movie would be given a rating, stored into an SQL server and then output to a website-based GUI, which was created using the python framework, Flask

In addition to this, I am currently working on a discord bot which allows for gambling games to be played using the service
