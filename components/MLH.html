Major League Hacking

MLH was a computing event which occured during october; allowing for anyone interested in the subject to join courses to learn more about specific areas.

Some of these subject areas included:

  • Programming
  • Python
  • Java
  • Conceptual areas
  • Blockchain
  • Cloud Computing
  • Website development
  • How to use industry-standard software
  • GitBash

Best event

During the course of the week, I had to do a large amount of activities. My favourite course out of the six was probably working with AWS.

The use of cloud computing is something that I have been curious about for a long time. In the case of making a website with it, I could see that AWS could be incredibly useful in the future. I hope to use it in conjunction with my discord bot to keep it hosted

In addition to learning about AWS, I also learnt more about how to use FLASK, a python framework (which I had used in an old project of mine), and also integration of MySQL with AWS

We were given a github repo and the resources required to make the website, and they taught us along the way. For instance, how to make an IAM account on AWS, as well as explaining all of the code they had used

During the week, I obtained 12.5 hours in MLH. Teaching me new skills in a variety of subjects
